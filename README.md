# Agent Deployment Templates

Docker and deployment templates for autonomous AI agent services.

## Templates

- **Node.js API Server** — Express-based API service for agent-to-agent commerce
- **MCP Server** — Model Context Protocol server for AI coding agent integration
- **Poll-based Worker** — Background task worker for marketplace platforms

## Quick Start

`ash
# Clone and pick a template
git clone https://github.com/eruens/agent-deployment-templates.git
cd agent-deployment-templates

# Deploy with Docker
docker compose up -d
`

## Stack
- Node.js 22
- TypeScript
- Docker / Docker Compose
- Health checks, metrics, and logging included
